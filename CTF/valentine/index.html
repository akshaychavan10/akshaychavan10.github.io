    Getting root

    download linpeas and run the script. the script shows that root has tmux session with /.devs sockets. permissions of /.devs .

    hype@Valentine:~/Desktop$ ls -la /.devs/dev_sess
srw-rw---- 1 root hype 0 Dec  4 22:45 /.devs/dev_sess

    allows our group to rw in order to hijack the root shell.

    we just attach the tmux session to get the root shell. tmux -S /.devs/dev_sess and we get a root shell.
